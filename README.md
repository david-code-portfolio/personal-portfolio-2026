# Personal Portfolio 2026 🚀

A modern, responsive personal portfolio website built to showcase my frontend development skills and help entrepreneurs grow their businesses through strategic web solutions.

## Features

- Clean, modern UI with smooth scroll animations
- Fully responsive across all screen sizes
- Light / Dark mode toggle
- Interactive components (live responsive preview, CRUD demo, data charts, search & filter)

## Built With

- [React](https://react.dev/) — UI library
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [Vite](https://vitejs.dev/) — Build tool & dev server
- [TailwindCSS](https://tailwindcss.com/) — Utility-first styling
